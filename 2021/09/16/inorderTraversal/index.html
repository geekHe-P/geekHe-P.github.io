二叉树中序遍历

题目描述

给定一个二叉树的根节点 root ，返回它的 中序 遍历。

示例1：

输入：root = [1,null,2,3]
输出：[1,3,2]

示例2：

输入：root = []
输出：[]

示例3：

输入：root = [1]
输出：[1]

示例4：

输入：root = [1,2]
输出：[2,1]

示例5：

输入：root = [1,null,2]
输出：[1,2]

提示：

  • 树中节点数目在范围 [0, 100] 内

  • -100 <= Node.val <= 100
